Everything that is outside the world of tags hangs off of window.rg

  • rg.map for all Google Map related functionality
  • rg.creditcard for all Credit Card validation functionality
  • rg.router if you have rg-router installed it assigns itself to rg

All the below cool helper functions with more to come...

RiotGear components make use of some common functionality that we thought you might like to use too.

Defined

  • rg.isDefined(thing) check if something is defined
  • rg.isUndefined(thing) check if something is undefined

Booleans

  • rg.toBoolean(thing) converts anything into true or false, handles 'true' as true and 'false as false
  • rg.isBoolean(thing) check if something is a Boolean

Numbers

  • rg.toNumber(thing) converts anything into a number, if NaN returns 0
  • rg.isNumber(thing) check if something is a Number

Promises and XHR

  • rg.isPromise(thing) checks for existance of .then()
  • rg.xhr(method, src, onload) will make a HTTP method call, to the src location. Calling onload once complete.

Misc

  • rg.isString(thing) check if something is a String
  • rg.isArray(thing) check if something is a Array
  • rg.isDate(thing) check if something is a Date
  • rg.isObject(thing) check if something is a Object
  • rg.isFunction(thing) check if something is a Function
  • rg.isRegExp(thing) check if something is a RegExp
  • rg.uid() will create a "unique" id by incrementing a number starting a 0
