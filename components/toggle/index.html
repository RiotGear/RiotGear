Use

Add this to your Riot app markup

<rg-toggle toggle="{ toggle }"></rg-toggle>

and this to your JavaScript

this.toggle = {
  checked: false,
  ontoggle: function (e) {
    console.log(e);
    _this.update(); // < this is important if you want your app to update on toggle
  }
};

Install

npm install riotgear-toggle --save
