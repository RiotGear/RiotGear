GA - Google Analytics

View code

There's nothing to visually see here but if you open developer tools and refresh the page you'll see network traffic that relates to Google Analytics.

Use

Add this to your Riot app markup

<rg-ga property="UA-12345678-1"></rg-ga>

Currently this component will only execute the basic Google Analytics pageview tag, i.e.

ga('send', 'pageview');

Future enhancements are coming that will allow events to be raised but for a lot of cases this is all you'll need.
