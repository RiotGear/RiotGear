Timepicker

View code

Use

Add this to your Riot app markup

<rg-timepicker time="{ new Date('Aug 10 2015') }"
               ampm="true"
               step="15"
               min="08:00"
               max="16:30"
               onopen="{ opened }"
               onclose="{ closed }"
               onselect="{ selected }">
</rg-timepicker>
this.opened = function () { ... }
this.closed = function () { ... }
this.selected = function (time) { ... }
  • time defaults to now if you don't specify one, accepts a date object
  • ampm toggles the 24 hours time format, the value is always hh:mm:ss though
  • step is the interval between options e.g. 15 produces 00:00, 00:15, 00:30, 00:45, 01:00...
  • min and max are the earliest and latest times within the available 24hrs

This component relies on rg-select from RiotGear, ensure you have either rg-select.min.js or rg.min.js referenced in your application.

Install

npm install riotgear-timepicker --save
