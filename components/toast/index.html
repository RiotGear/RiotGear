Toasts

Use

HTML 

<rg-toast toasts="{ toasts }"></rg-toast>

JavaScript 

this.toasts = new RgToasts({
  position: 'bottomright',
  toasts: [{
    content: 'You have received an update to something interesting',
    sticky: true,
    timeout: 6000,
    onclick: function (e) {...},
    onclose: function () {...}
  }]
})

Options

  • position available positions are topright, bottomright, bottomleft and topleft
  • isvisible show and hide the entire toast container
  • toast.content what you want to say in the popup box, supports HTML
  • toast.sticky if true will turn off the timeout and the toast will remain on the page until clicked on
  • toast.timeout use to override the delay (in ms) before it disappears, default is 6 seconds
  • toast.isvisible show and hide the toast itself
  • toast.onclick is called when the toast is clicked on
  • toast.onclose fires when the toast disappears, either by clicking on it or via timer
