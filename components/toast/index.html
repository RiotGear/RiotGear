Toast

View code

Use

Add this to your Riot app markup

<rg-toast toasts="{ toasts }" position="bottomright"></rg-toast>

The available positions are topright, bottomright, bottomleft and topleft.

and this to your JavaScript

this.toasts = [{
  text: 'You have received an update to something interesting',
  sticky: true,
  timeout: 6000,
  onclick: function (e) {...},
  onclose: function () {...}
}];

The toast notification can be clicked on after which it will disappear.

  • text what you want to say in the popup box
  • sticky if true will turn off the timeout and the toast will remain on the page until clicked on
  • timeout use to override the delay (in ms) before it disappears, default is 6 seconds
  • onclick is called when the toast is clicked on
  • onclose fires when the toast disappears, either by clicking on it or via timer

Install

npm install riotgear-toast --save
