Use

Add this to your Riot app markup

<rg-loading show="{ loading }">
  Please wait...
</rg-loading>

and this to your JavaScript

this.loading = false;

To make the loading overlay appear or disappear simply toggle the show property.

this.loading = true|false;

The spinner is an integrated SVG animated icon.

You can use either the show or if Riot conditional attributes or even the dynamic class attribute to display the loading panel.

The loading overlay is specific to the containing element but this can be overridden in CSS if you want it to cover the whole page.
