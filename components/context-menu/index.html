Context Menu

View code

Right click anywhere on this page to see the menu appear.

Use

HTML 

<rg-context-menu menu="{ menu }"></rg-context-menu>

Then assign the context menu to the elements you would like to by using rg-context-menu

<div rg-context-menu="myMenu">
  ... your page content ...
</div>

JavaScript 

this.menu = new RgContextMenu({
  name: 'myMenu',
  items: [{
    content: '<strong>View source</strong>',
    onselect: itemSelected
  }, {
    content: 'Save as...',
    inactive: true,
    onselect: itemSelected
  }, {
    content: 'Reload',
    onselect: itemSelected
  }],
  onopen: function () {},
  onclose: function () {}
})

Options

  • name the target menu to hook into
  • onopen will fire when the menu appears
  • onclose is the array of items to display in the popup list
  • items is the HTML content of the item
  • items.content is the HTML content of the item
  • items.inactive set a inactive class and prevents events
  • items.onclick fires when an item is clicked on. The menu will also automatically close.

Methods

  • add(item) adds an item to the list of options
  • select(item) selects an item in the list
