Behold

View code

Use

Add this to your Riot app markup

<rg-behold image1="img/first.jpg" image2="img/second.jpg" mode="swipe"></rg-behold>

There are 2 options available:

  • image1 the path to the first image to display
  • image2 the path to the second image to compare to the first
  • mode can be either swipe or fade, default to fade

Install

npm install riotgear-behold --save
