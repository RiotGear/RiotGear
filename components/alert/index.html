Alert

View code

Use

Add this to your Riot app markup

<rg-alert alerts="{ alerts }"></rg-alert>

and add this to your JavaScript

this.alerts = [{
  type: 'danger',
  msg: 'Danger! Something bad happened.',
  onclose: function (e) {}
}];

To add alerts simply push alerts on to the array.

Install

npm install riotgear-alert --save
