Placeholdit

View code

Use

Add this to your Riot app markup

<rg-placeholdit width="200"
                height="100"
                background-color="1fadc5"
                color="4df"
                font-size="50"
                text="JPEG"
                format="jpg">
</rg-placeholdit>
  • width and height take the number of pixels you want the image to be
  • background-color is the HEX value of the background you want
  • color is the HEX value of the text color
  • font-size is the size of the text
  • text is the text to display, if none specified it will default to the dimensions
  • format can be gif, png, jpeg or jpg. It defaults to png

This component uses placehold.it to render the images.

Install

npm install riotgear-placeholdit --save
