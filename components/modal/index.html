Modal

View code

Use

Add this to your Riot app markup

<rg-modal modal="{ modalOptions }">
  Modal body text
</rg-modal>

and this to your JavaScript

this.modalOptions = {
  heading: 'Modal heading',
  visible: false,
  buttons: [
    { action: function () {}, text: 'Save' },
    { action: function () {}, text: 'Cancel', style: 'color: cornflowerblue;' }
  ],
  onclose: function (e) {}
};

To make the modal appear or disappear simply toggle the modal's visible property.

this.modal.visible = true|false;
