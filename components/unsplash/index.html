Unsplash

View code

Use

Add this to your Riot app markup

<rg-unsplash width="200"
             height="100"
             greyscale="true"
             random="true"
             blur="true"
             image="491"
             gravity="north|south|east|west">
</rg-unsplash>
  • width and height take the number of pixels you want the image to be
  • greyscale or grayscale will make the image black and white
  • random will randomise the image on reload
  • blur will apply a blur filter to the image
  • image can be used to specify a particular image
  • gravity is the cropping gravity

This component uses unsplash to render the images.

Install

npm install riotgear-unsplash --save
