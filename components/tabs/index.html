Tabs

View code

Use

HTML 

<rg-tabs tabs="{ tabs }"></rg-tabs>

JavaScript 

this.tabs = new RgTabs({
  tabs: [{
    heading: 'Tab <em>one</em>',
    content: 'This is tab one'
  }, {
    heading: 'Tab two',
    content: 'This is tab two',
    active: true
  }, {
    heading: 'Disabled tab',
    content: 'This is disabled tab',
    disabled: true
  }, {
    heading: 'Tab three',
    content: 'This is tab three content',
    include: new RgInclude({
      url: 'tab.html',
      unsafe: true
    })
  }]
})

Options

  • tabs an array of tab objects
  • tab.heading HTML content of the tab top
  • tab.content HTML content of the tab body
  • tab.active whether the tab is selected and visible
  • tab.disabled whether or not the tab can be active
  • tab.include an RgInclude object that will inject the contents of a separate file into the tab body
  • onopen callback when a tab is selected

Methods

  • select(tab) switch tabs
