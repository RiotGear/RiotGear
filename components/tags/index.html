Use

Add this to your Riot app markup

<rg-tags items="{ countries }"
         tags="{ tags }"
         type="text"
         value="Canada"
         placeholder="Enter a country name"
         onselect="{ ... }"
         onclose="{ ... }"
         onfilter="{ ... }">
</rg-tags>

If you want to populate the autocomplete list assign an array to the items attribute

this.countries = [{
  text: 'England'
}, {
  text: 'Scotland'
}, {
  text: 'Wales'
}, {
  text: 'Ireland'
}];

If you want to pass a list of already selected tags assign an array to the tags attribute

this.tags = [{
  text: 'America'
}];
  • onselect fires when an item is selected from the list and passes the item to the callback
  • onclose is called when the dropdown is closed
  • onfilter fires everytime something is entered into the text box or when the box gains focus

npm install riotgear-tags --save
