Select

Use

HTML 

<rg-select select="{ select }"></rg-select>

JavaScript 

this.select = new RgSelect({
  autocomplete: true,
  placeholder: 'Please select a card',
  filterplaceholder: 'Filter cards',
  filteron: 'text',
  onopen: function () {},
  onclose: function () {},
  onfilter: function () {},
  onselect: function () {},
  options: [{
    text: 'England'
  }, {
    text: 'Scotland'
  }, {
    text: 'Wales'
  }, {
    text: 'Ireland'
  }]
})

Options

  • options an array of objects to populate the dropdown list with
  • option.text the text to display in the list
  • autcomplete toggles the autocomplete / select mode, making the textbox readonly/editable
  • placeholder placeholder text when nothing selected
  • hasfilter toggle the filter visibility
  • filterplaceholder placeholder text in the filter box when nothing entered
  • filteron the property on the options objects to search when filtering
  • onselect fires when an item is selected from the list and passes the item to the callback
  • onclose is called when the dropdown is closed
  • onfilter fires everytime something is entered into the text box or when the box gains focus

Methods

  • open() make the dropdown visible
  • close() hide the dropdown
  • toggle() toggle the dropdown visibility
  • filter(text) add text to the input and filter the list
  • select(item) select an item and put the item text in the input
