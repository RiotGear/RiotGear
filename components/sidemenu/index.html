Sidemenu

View code

Use

The sidemenu now extends the functionality of the drawer tag but the position is fixed to the left hand side. If you want more control please use the drawer tag.

HTML 

<rg-sidemenu sidemenu="{ menu }"></rg-sidemenu>

JavaScript 

this.menu = new RgSidemenu({
  header: 'Select an option',
  isvisible: true,
  onopen: function (item) {},
  onclose: function (item) {},
  onselect: function (item) {},
  items: [{
    content: 'Open in Safari',
    action: function () {},
    active: true
  }]
})

Options

  • header the text at the top of the menu
  • onopen callback when the menu is opened using the open method
  • onclose callback when the menu is closed using the close method
  • onselect callback when an item is select using the select method
  • isvisible toggle this to show or hide the menu
  • items array of menu items
  • item.content the HTML to display
  • item.action the callback when the item is selected
  • item.active whether the item is in an active state or not

Methods

  • open() open the menu
  • close() close the menu
  • toggle() toggle the visibility of the menu
  • select(item) select an item
