Use

Add this to your Riot app markup

<rg-include src="include_this.html" async="false" unsafe="true"></rg-include>

The component will then request the file at the specified src path and insert the contents into the page.

Psst. The contents of this page is being inserted via rg-include. Take a look at the source ;-)

There are 2 options available.

  • src the path to the file you want to include. This is required.
  • unsafe to render HTML as HTML rather than the escaped set this to true. By default it is turned off.
