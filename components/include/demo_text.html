Use

HTML 

<rg-include include="{ include }"></rg-include>

JavaScript 

this.include = new RgInclude({
  url: 'demo.html',
  unsafe: true,
  async: false
})

The component will then request the file at the specified url path and insert the contents into the page.

Psst. The contents of this page is being inserted via rg-include. Take a look at the source ;-)

Options

  • url the path to the file you want to include
  • unsafe to render HTML as HTML rather than the escaped set this to true
  • async whether the network request should be executed asynchronously
