Get the code

There are a few ways of grabbing the code.

Install via NPM

RiotGear has been published to NPM. 

npm install riotgear --save

Or alternatively:

Install via Bower

RiotGear is also available via Bower. 

bower install riotgear

Or finally you could:

Download the source

Both versions of the source are available to download:
» riotgear.js
» riotgear.min.js

Individual Components

Each component is also available from NPM, Bower and as a download file.
