Get the code

There are a few ways of grabbing the code.

CDN

RiotGear is hosted by JSDelivr

» https://cdn.jsdelivr.net/riotgear/1.17.1/rg.min.js (v1.17.1)
» https://cdn.jsdelivr.net/riotgear/latest/rg.min.js (latest) 

<script src="//cdn.jsdelivr.net/riotgear/1.17.1/rg.min.js"></script>

Install via NPM

RiotGear has been published to NPM. 

npm install riotgear --save

Or alternatively:

Download the source

Both versions of the source are available to download:
» rg.js
» rg.min.js

Individual Components

Each component is also available from NPM and as a download file.
