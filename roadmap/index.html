Roadmap

Are we there yet?!

Long term goals

RiotGear is here to help you build websites. We want to help you increase your development speed. To do that we're planning on developing RiotGear further. First stop...

More Components

We're always adding more tags. The more tags we have the less work you have to do to build your user interfaces. Anyone can see what components we are going to add, and anyone can request new components too.

Themes

As part of the v2 update all the tags are now much more customisable with CSS allowing for themable HTML markup that can handle flexible CSS rules and animations.

With this new found ability we are taking pull requests for themes that anyone would like to make. We might build some themes ourselves but we think we have more important work to do for the time being...

RiotGear Framework

At the moment RiotGear is simply a library of useful components. But they're a little disjointed. We need something to bring them all together, something to give RiotGear a better identity and purpose.

As such we are planning on developing RiotGear into a mobile first UI framework covering all the basics you need to build a fully responsive website, akin to Bootstrap, Semantic, Foundation, but with RiotGear components at its heart.

More...

We like to look towards the future, the further ahead we look the more focus we have on the present. But things will change so the above is not a promise simply our current state of mind.

As always if you have any ideas or suggestions please:

  • Let us know in the forum
  • Raise an issue in GitHub
  • or send a tweet
